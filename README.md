# 🔐 LegalGuard – AI-Powered Terms & Conditions Analyzer

## 🚀 Overview

LegalGuard is an AI-based web application that helps users understand complex Terms & Conditions before accepting them. Instead of blindly clicking "I Agree", users get a clear breakdown of what they are signing up for.

---

## ❌ Problem

Most users do not read or understand lengthy legal documents, which leads to:

* Hidden data sharing
* Auto-renewal traps
* Unclear liabilities

---

## ✅ Solution

LegalGuard analyzes T&C documents and provides:

* 🧠 Simplified explanations
* 🔴 Risk classification (High / Medium / Low)
* 📊 Overall risk score
* 📌 Key highlights (data usage, payments, permissions)
* ⚠️ Smart recommendation: Accept or Review

---

## ⚙️ Features

* Upload or paste Terms & Conditions
* Clause-by-clause breakdown
* Highlight risky sections
* Easy-to-read summary
* User-friendly UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **AI Engine:** NLP / LLM-based analysis

## ▶️ How to Run

1. Clone the repository
2. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```
3. Run the app

   ```bash
   python app.py
   ```
4. Open in browser

   ```
   http://127.0.0.1:5000
   ```
